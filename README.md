# ExpressVPN Landing Page (Frontend)

## Descripción del proyecto
Este proyecto consiste en la recreación de una **landing page inspirada en el diseño del sitio web de ExpressVPN**.

El objetivo de la actividad fue practicar la **estructura semántica en HTML y el diseño con CSS**, replicando la apariencia visual del sitio.

El proyecto corresponde únicamente al **desarrollo del frontend**, por lo que **no incluye funcionalidades dinámicas ni conexión con backend**.

---

## Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo `index.html` en cualquier navegador web.

No se requiere instalación de dependencias.
